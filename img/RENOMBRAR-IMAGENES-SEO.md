# 📸 Guía de nombrado SEO — La Fábrica de Sonrisas
## Cómo renombrar todas las imágenes antes de subirlas

---

### REGLAS GENERALES

- Todo en **minúsculas**
- Palabras separadas por **guiones** (nunca espacios ni guiones bajos)
- Sin tildes ni caracteres especiales (ñ → n, é → e, etc.)
- Incluir siempre **keyword + ubicación** cuando aplique
- Formato: `[descripcion]-[contexto]-[ubicacion]-[numero].jpg`

## 📋 CÓMO RENOMBRAR EN BLOQUE (Mac / Windows)

### En Mac:
1. Selecciona todas las fotos de una carpeta
2. Clic derecho → "Renombrar X elementos"
3. Elige "Formato" → escribe el nombre base del grupo
4. Añade numeración desde 01

### En Windows:
1. Selecciona todas → F2
2. Escribe el nombre base → Enter (Windows añade (1), (2)... automáticamente)

### Mejor opción (cualquier sistema):
Usa **Bulk Rename Utility** (gratis) o **NameChanger** (Mac) para renombrar en lote con control total.

## ✅ CHECKLIST ANTES DE SUBIR CADA IMAGEN

- [ ] Nombre en minúsculas con guiones
- [ ] Contiene keyword principal (cumpleanos / fiestas / pozuelo)
- [ ] Sin tildes ni ñ
- [ ] Formato JPG (fotos) o WebP si es posible
- [ ] Peso máximo recomendado: **200KB por imagen** (usa squoosh.app para comprimir)
- [ ] Resolución mínima: 800 × 600 px
