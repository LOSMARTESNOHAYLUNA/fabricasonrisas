# 🎈 La Fábrica de Sonrisas — Landing Page

Landing page profesional para La Fábrica de Sonrisas, espacio de celebraciones infantiles en Pozuelo de Alarcón, Madrid.

## 🚀 Stack

- HTML5 semántico
- CSS3 (variables, animaciones, responsive mobile-first)
- JavaScript vanilla (sin dependencias)
- Google Fonts (Fraunces + Plus Jakarta Sans)
- Deploy en Vercel

## 📁 Estructura


## ⚙️ Deploy en Vercel

1. Sube este repo a GitHub
2. Ve a [vercel.com](https://vercel.com) → New Project → Import desde GitHub
3. Framework Preset: **Other**
4. Root Directory: `/`
5. Build Command: (vacío)
6. Output Directory: (vacío)
7. Click **Deploy**
